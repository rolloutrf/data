# Собрать платёжный экраны вывода

[DESGM-86](https://xn--80abj3bk.xn--80avakjqg.xn--p1ai/workbench/rollout/tracker/DESGM-86)

К качестве основы можно взять экран оплаты чего-ли, только помимо селекта «Куда» переводим, нужно добавить «Откуда» и «сколько» переводим (можно сделать в виде инпута для ввода суммы и чипсов для саджестов)

Также нужно предусмотреть экран, на котором должен располагаться умный инпут с вводом какого, либо реквизита: Номера, карты, счета или скана камеры + отображение контактов из записной книжки
