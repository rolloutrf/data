## 1. Чем занимается ваша компания (одно предложение):

РОЛЛАУТ™ — это опенсорс-платформа для цифровой торговли и финтеха, которая объединяет экспертов и компании для создания эталонных решений с фокусом на Открытые API




## 2. Основная ценность:

Демократизация качественных торгово-финансовых решений и устранение барьеров для их масштабирования за счёт открытых стандартов и коллаборации.




## 3. Цель/миссия:

Сформировать индустриальную open-source платформу для торгово-финансовой экономики, которая позволит бизнесам и разработчикам быстро запускать совместимые сервисы, снижая стоимость входа и ускоряя инновации.




## 4. Инструменты для достижения целей:

- Продуктовые спецификации;

- Дизайн-системы и UI-киты;

- Коллективная разработка через комьюнити;

- Партнёрства с экосистемами, банками и отраслевыми игроками.




Продукт / рынок




1. О продукте одним предложением:

РОЛЛАУТ™ — это набор открытых модулей и стандартов (API, UI, CJM, Спецификации) для построения цифровых торгово-финансовых сервисов нового поколения.




2. Цель продукта:

Сократить время и стоимость разработки и интеграции финтех функций, предоставив отечественным площадкам готовые блоки для интеграции и масштабирования.




3. Самые важные 3 вещи:

Детализированные спецификации по каждой из функций;

Стандартизированыне интерфейсы и API для торговых платформ.

Централизованная картина всех платформенных компаний и её участников (для развития партнерств);




4. Самые не важные 3 вещи:

Закрытая кастомизация под конкретную компанию;

Агрессивный маркетинг ради трафика;

Фокус на рекламной монетизации.




5. Чем отличается от аналогов:

Open Source (в отличие от проприетарных решений банков и экосистем);

Design First подход с качественным UI и эталонным UX;

Система совместимых друг-с-другом модулей по открытым стандартам Open API.




6. Краткосрочные цели:

Сбор и активизация комьюнити.

Черновик со спецификациями по всем 12 функциям

UI-библиотеки ключевых модулей и интерфейсная витрина со всеми вариациями платформы

Подготовленные промо материалы

Контент для популяризации




7. Долгосрочные цели:

Техническая реализация со всеми модулями

Партнёрская сеть с экосистемами и банками;

Пользователи / клиенты




1. Целевая аудитория:

Эксперты и индивидуальные контрибьюторы бигтеха и дизайн-агентств — в качестве контрибьюторов;

Менеджеры, руководители лиды продуктовых и технических функций миддл теха — в качестве потребителей.




2. Типичный клиент:

Отраслевая платформа в одной из ключевых ниш ФудТех, Услуги, Косметика, .. , которая хочет запустить платежный сервис или финансовый модуль, но не хочет тратить миллионы на разработку с нуля.




3. Откуда будут приходить клиенты:

Работа со стороны функции партнерств;

Telegram-комьюнити (Блокчейн и Деньги / Финтех Кружок);

Публикации на VC.

4. Их проблемы:

5. Чем отличаемся:

6. Почему выберут нас:

Возможность влиять на развитие продукта.




7. Что хотим взять у конкурентов:
